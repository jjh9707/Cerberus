프롬프트:
청소년을 위한 사이버 보안 교육 웹사이트 "필터온(FilterOn)"을 만들어주세요.
핵심 요구사항:

타겟: 초등학생~중학생 (어린이, 청소년)
디자인:

밝고 친근한 색상 (파랑, 초록, 주황 등)
큰 버튼과 아이콘으로 직관적인 UI
모바일 친화적인 반응형 디자인
게임 같은 느낌의 인터페이스


주요 기능:
A. 홈 화면

웹사이트 소개
학습 모듈 선택 버튼들
현재 보유 금액 표시 (화면 상단에 크게, 예: "💰 100,000원")

B. 가상 머니 시스템 (핵심!)

초기 금액: 100,000원
문제를 틀릴 때마다 금액 차감
차감 금액은 문제 중요도에 따라 다름:

위험도 낮음: -5,000원 (예: 단순 스팸)
위험도 중간: -15,000원 (예: 스미싱 링크 클릭)
위험도 높음: -30,000원 (예: 계좌이체 요구, 개인정보 입력)
위험도 매우 높음: -50,000원 (예: 가족 사칭 긴급 송금)


돈이 0원이 되면 "파산" 화면 표시
정답을 맞춰도 돈은 늘어나지 않음 (현실처럼 조심해야만 지킬 수 있음)

C. 게임형 퀴즈 모듈

실제 피싱 메시지 vs 정상 메시지 구별 퀴즈
각 문제마다:

메시지 내용 표시
"안전해요" / "위험해요" 선택 버튼
시간 제한 (30초)
문제 위험도 표시 (예: ⚠️ 위험도: 높음)


문제 예시:

"엄마야, 급해. 지금 바로 여기로 10만원 보내줘" [위험도: 매우 높음, -50,000원]
"택배가 도착했습니다. 링크 클릭하세요 hxxp://..." [위험도: 높음, -30,000원]
"무료 게임 아이템 받으세요!" [위험도: 중간, -15,000원]
"친구 민수: 내일 학교에서 보자!" [정상 메시지]


틀리면:

화면에 "-30,000원" 같은 빨간 텍스트 애니메이션
현재 잔액 업데이트
해설 표시: "이 메시지는 OO 사기예요. 실제로 속으면 이만큼 피해를 볼 수 있어요!"


맞히면:

"잘 막았어요! 돈을 지켰어요!" 메시지
해설 표시



D. 학습 모듈 (3가지)

스미싱 모듈: 문자 메시지 사기 식별법
SNS 사칭 모듈: 가짜 친구 계정 구별법
게임 아이템 사기 모듈: "무료 아이템" 광고 주의법


각 모듈마다 설명 페이지 + 실습 퀴즈 (가상 머니 적용)

E. 딥보이스 체험 페이지 (간단하게만 구성)

설명 섹션:

"AI 딥보이스 기술이란?"
"최근 AI가 부모님이나 친구 목소리를 그대로 따라해서 사람들을 속이는 범죄가 늘어나고 있어요"
"목소리만 믿어서는 안 돼요. 꼭 직접 확인해야 해요!"


체험 안내 버튼/섹션:

"딥보이스 체험하기" 버튼 (클릭 시 안내 메시지)
안내 메시지: "딥보이스 체험 기능은 곧 추가될 예정이에요. 여러분의 목소리를 녹음해서 다른 문장을 말하는 것처럼 들려드릴 거예요!"


개인정보 보호 안내 (중요!):

눈에 잘 띄는 박스 또는 강조 영역에 표시
"🔒 안전 약속: 여러분의 목소리는 절대 저장되지 않아요!"
"녹음된 목소리는 체험이 끝나면 바로 삭제되며, 서버에 전송되거나 보관되지 않습니다"
"필터온은 여러분의 개인정보를 소중히 지킵니다"



F. 결과 페이지

최종 남은 금액 표시 (크게)
결과 메시지:

80,000원 이상: "훌륭해요! 사이버 보안 전문가예요!"
50,000~79,999원: "잘했어요! 조금만 더 조심하면 완벽해요!"
20,000~49,999원: "아쉬워요. 다시 도전해서 더 많이 지켜봐요!"
0~19,999원: "위험해요! 다시 배워서 꼭 돈을 지켜내요!"


"다시 시작하기" 버튼 (100,000원으로 리셋)


기술 스택:

HTML, CSS, JavaScript
로컬 스토리지로 현재 금액/진행도 저장
애니메이션 효과 (CSS transitions, 특히 금액 차감 시)


말투와 문구:

복잡한 용어 대신 쉬운 표현 사용
"피싱" → "이상한 메시지", "위험한 문자"
친근하고 격려하는 톤
금전적 피해를 실감할 수 있는 메시지 추가

"이런 메시지에 속으면 실제로 이만큼 손해를 볼 수 있어요"
"조심하지 않으면 용돈이 다 사라질 수 있어요"




추가 요청:

처음 방문자를 위한 간단한 튜토리얼
금액 차감 시 효과음/진동 효과 (선택사항)
학부모용 안내 페이지 추가



중요 사항:

점수 시스템, 총점, 배지 시스템은 모두 제외
오직 가상 머니 시스템만 사용
딥보이스 체험은 간단한 안내 페이지로만 구성 (실제 구현은 나중에 따로 추가 예정)
개인정보 보호 문구 필수 포함

완성도 있고 실제로 작동하는 웹사이트를 만들어주세요!
