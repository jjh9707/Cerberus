기존 웹사이트에 딥보이스(DeepVoice) 체험 페이지를 수정해주세요.
준비 중이라고 했던 체험 시작하기 버튼을 활성화 시키고, 내부 페이지를 생성해 주세요.
핵심 요구사항:

페이지 구조:

기존 웹사이트의 디자인과 통일된 스타일

딥보이스 체험 기능 (3단계 프로세스):
[1단계] 목소리 녹음

큰 "🎤 녹음 시작" 버튼
녹음 중일 때: 빨간 점 애니메이션 + "녹음 중..." 표시 및 대본 표시
녹음 시간 표시 (최대 20초)
대본: [
계절이 지나가는 하늘에는
가을로 가득 차 있습니다.
 
나는 아무 걱정도 없이
가을 속의 별들을 다 헤일 듯합니다.
 
가슴 속에 하나 둘 새겨지는 별을
이제 다 못 헤는 것은
쉬이 아침이 오는 까닭이요,
내일 밤이 남은 까닭이요,
아직 나의 청춘이 다하지 않은 까닭입니다.
 
별 하나에 추억과
별 하나의 사랑과
별 하나에 쓸쓸함과
별 하나에 동경과
별 하나에 시와
별 하나에 어머니, 어머니, ]

"⏹️ 녹음 종료" 버튼
녹음된 오디오 재생 버튼 ("내 목소리 들어보기")

[2단계] 문장 선택

제목: "AI가 말하게 할 문장을 골라보세요"
3개의 문장 선택 버튼:

"엄마, 누가 쫓아와! 살려줘!"
"아는 삼촌이 맛있는 거 사준대. 같이 갈래?"
"엄마 차에 강아지 있는데 볼래?"
선택된 문장 강조 표시

[3단계] AI 변환 및 재생

"🤖 AI로 변환하기" 버튼
로딩 애니메이션 (2-3초)
변환 완료 후: "🔊 변환된 목소리 듣기" 버튼
재생 시 파형 애니메이션 또는 스피커 아이콘 애니메이션


경고 및 교육 메시지 (페이지 상단과 하단에 눈에 띄게 표시):
상단 안내:

   ⚠️ 딥보이스 기술 체험
   
   AI가 여러분의 목소리를 학습해서 
   말하지 않은 문장을 여러분 목소리로 만들어요.
   
   이 기술이 나쁜 사람들에게 사용되면 
   부모님이나 친구를 사칭할 수 있어요!
   
   → 목소리만 믿지 말고 꼭 직접 확인하세요!
하단 개인정보 보호 안내 (중요!):
   🔒 안전 약속
   
   ✅ 녹음된 목소리는 절대 저장되지 않아요
   ✅ 서버에 전송되거나 보관되지 않아요
   ✅ 체험이 끝나면 즉시 삭제돼요
   ✅ 브라우저에서만 실행되고 외부로 나가지 않아요
   
   필터온은 여러분의 개인정보를 소중히 지킵니다

기술 구현:
녹음 기능:

Web Audio API의 MediaRecorder 사용
사용자 마이크 권한 요청
녹음 데이터를 Blob으로 저장 (메모리에만, 서버 전송 X)

AI 변환 (간소화 버전):

실제 딥보이스 API는 나중에 통합 예정이므로, 현재는 시뮬레이션 모드로 구현
2-3초 로딩 후, 미리 준비된 샘플 오디오 재생 (또는 원본 녹음에 약간의 효과를 추가)
나중에 실제 TTS API (ElevenLabs, Google TTS 등) 통합 가능하도록 구조화

데이터 관리:

모든 오디오 데이터는 로컬 메모리에만 저장
페이지 새로고침 시 모든 데이터 삭제
localStorage나 서버에 절대 저장 금지
사용자가 "다시 하기" 버튼 누르면 이전 녹음 완전 삭제


추가 UI 요소:

각 단계마다 진행 상태 표시 (1/3, 2/3, 3/3)
"다시 녹음하기" 버튼
"처음으로 돌아가기" 버튼
체험 완료 후 메시지:

     체험 완료! 😮
     
     무섭지 않나요?
     AI 기술이 이렇게 실제 목소리를 똑같이 만들 수 있어요.
     
     그래서 전화나 음성 메시지만으로는
     누가 진짜인지 알 수 없어요!
     
     💡 중요: 목소리만 믿지 말고 꼭 직접 확인하세요!

에러 처리:

마이크 권한 거부 시: "마이크 권한이 필요해요. 브라우저 설정에서 허용해주세요!"
녹음 실패 시: "녹음에 실패했어요. 다시 시도해주세요."
너무 짧은 녹음 (2초 미만): "최소 3초 이상 녹음해주세요!"


디자인 가이드:

버튼은 크고 눈에 잘 띄게
경고 메시지는 노란색/주황색 배경
개인정보 보호 안내는 초록색 배경
각 단계마다 명확한 구분선
아이콘 활용 (🎤 🔊 🤖 ⚠️ 🔒)


접근성:

큰 폰트 사이즈
명확한 버튼 라벨
진행 상황을 시각적으로 명확하게 표시



중요:

실제 AI 음성 변환은 나중에 API 통합 예정
현재는 교육 목적의 시뮬레이션으로 구현
개인정보 보호 메시지는 반드시 포함
모든 오디오는 로컬에서만 처리, 절대 서버 전송 금지

완성도 있고 작동하는 딥보이스 체험 페이지를 만들어주세요!
